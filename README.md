# instagram-clone

Backend(Node + Express + Mongodb) Frontend(React + Tailwindcss)

Các tính năng hiện tại của trang web
- Sign up / Sign in
- Change Avatar
- Follow / Unfollow
- Create Post
- Like / Dislike post
- Comment post

Những công nghệ được sử dụng
- Create react app
- React router v6
- Tailwindcss
- Axios
- Firebase
- Express rest api
- Jsonwebtoken
- Mongoose
